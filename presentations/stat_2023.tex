\begin{frame}{Дисперсійний аналіз}
\begin{block}{}
Дисперсійний аналіз~--- це тест, який використовується, щоб з'ясувати, чи існує статистично значуща різниця між середніми значеннями більш ніж однієї групи.
\end{block}

\includegraphics[scale=0.28]{anova_table.jpg}

\end{frame}







\begin{frame}{Приклад}

\includegraphics[scale=0.9]{example1.png}

\begin{block}{}
Середні значення: $\overline{X_{1}} = \frac{25+30+28+36+29}{5} = 29.6$, $\overline{X_{2}} = 45$, $\overline{X_{3}} = 31.2$, $\overline{X_{4}} = 60$, загальне $X = \frac{25+...+73}{20} = 41.45$.

Сума квадратів між групами: $SSB_{c} = \sum n\times(\overline{X_{i}} - \overline{X} )^{2} = 5(29.6-41.45)^{2} + 5(45-41.45)^{2} + 5(60-41.45)^{2} + 5(31.2-41.45)^{2} = 3010.95$

Сума квадратів в межах груп: $SSW = (25-29.6)^{2} + (30-29.6)^{2} + ... + (73-41.45)^{2} = 918$

Загалом: $\textmd{Total} = SSB_{c} + SSW = 3928.95$.

Замірів: $N=20$, $df_{w} = (5-1)*4 = 16$, $df_{b} = 4-1 = 3$, $df_{t}=5*4-1 = 19$;

Далі, $MS_{b} = \frac{SSB_{c}}{df_{b}} = 1003.65$, $MS_{w}=\frac{SSW}{df_{w}} = 57.37$, $f = \frac{1003.65}{57.375} = 17.5$


\end{block}

\end{frame}






\begin{frame}{F-test}

\includegraphics[scale=0.46]{f_table.jpg}

\begin{block}{}
$f = 17.5 > 5.29$~--- існує різниця між середніми значеннями.


\end{block}
\end{frame}







\begin{frame}{Аналіз із повторними вимірами}
\begin{block}{}
Аналіз із повторними вимірами~--- використовується для визначення того, чи існує статистично значуща різниця між середніми значеннями трьох або більше груп, у яких однакові суб’єкти з’являються в кожній групі.
\end{block}

\end{frame}
