\section*{Остачі та китайська теорема про остачі}


\begin{large}

1. Знайдіть остачу від ділення числа $4^{1000}$ на $7$.

\textbf{Китайська теорема про остачі.} \textit{Нехай $a_{1},...,a_{n}$~--- попарно прості натуральні числа, то для довільних натуральних $r_{1},...,r_{n}$, таких, що $0\leq r_{i}<a_{i}$ для кожного $i$, $0\leq i\leq n$, існує єдине число $x$, таке, що $x\equiv r_{i}(\textmd{mod }a_{i})$.}

2. Знайти найменше натуральне число, яке дає остачі 1,2,3,4 по модулям 5,7,11,12.

3. Про число $x$ відомо, що $x^{2}$ закінчується на 001. Якими можуть бути останні три цифри числа $x$.

4. Знайдіть всі трицифрові $x$, що останні 3 цифри $x^{2}$ співпадають з $x$.

5. Довести, що для взаємнопростих $m$ та $n$ рівність $a\equiv b(\textmd{mod }mn)$ рівносильна системі рівностей $a\equiv b(\textmd{mod }n)$ та $a\equiv b(\textmd{mod }m)$.


\end{large}
