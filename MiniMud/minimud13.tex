\section*{Доведення нерівностей.}

\quad Для додатніх $a$, $b$, $c$, $d$ доведіть нерівності

1. $a^{2}+b^{2}+c^{2}\geq ab+bc+ca$.

2. $a^{4}+b^{4}+c^{4}+d^{4}\geq 4abcd$.

3. $a(1+b)+b(1+c)+c(1+a)\geq 6\sqrt{abc}$.

4. $(a+b)(b+c)(b+c)\geq 8abc$.

5. $(1+ab)^{2}+(1+cd)^{2}+(ac)^{2}+(bd)^{2}\geq 1$.

6. Розв`яжіть рівняння $a^{2}+b^{2}+c^{2}+d^{2}-ab-bc-cd-d+\frac{2}{5}$

7. Доведіть нерівність $(\frac{a}{4})^{2}+b^{2}+c^{2}\geq ab-ac+2bc$.

8. Доведіть для довільних $x,y,z$ нерівність $$x^{4}+y^{4}+z^{2}+1\geq x(xy^{2}-x+z+1).$$

\section*{Використання математичної індукції для доведення нерівностей.}

\quad 9. Для $n\geq3$ доведіть нерівність: $$\frac{1}{n+1}+\frac{1}{n+2}+...+\frac{1}{2n}>\frac{3}{5}.$$

10. Яке з чисел $2^{2^{.^{.^{.^{2}}}}}$ (10 двійок) чи $3^{3^{.^{.^{.^{3}}}}}$ (9 трійок) більше?

11. Для $n\geq1$ доведіть нерівність: $3^{n}>n\cdot 2^{n}$.

12. Для $n\geq4$ доведіть нерівність: $2^{n}<n!$.

13. При яких $n$ виконується нерівність: $2^{n}\geq n^{2}$?
