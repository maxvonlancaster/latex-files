\section*{Комплексні числа.}
\end{center}


\textbf{Формула Муавра}: для $z=r(\cos \varphi + i\sin \varphi)$ справджується: $$z^{n}=r^{n}(\cos n\varphi + i\sin n\varphi)$$

\begin{enumerate}
\item Довести, що для довільних двох чисел, кожне з яких є сумою двох квадратів, їхній добуток також є сумою двох квадратів.

\item Знайти всі натуральні числа $n>1$ такі, що $x^{n}+x^{2}+1$ ділиться на $x^{2}+x+1$.

\item При яких $p$ і $q$ многочлен $x^{4}+1$ ділиться на $x^{2}+px+q$.

\item Обчисліть $\cos \frac{2\pi}{7}+\cos \frac{4\pi}{7}+\cos \frac{6\pi}{7}$

\item Нехай $\sin \alpha = \frac{3}{5}$. Доведіть, що $\sin 25\alpha$ має вигляд $\frac{n}{5^{25}}$, де $n$~--- ціле число, що не ділиться на 5.

\item Доведіть, що $\cos \varphi + ... + \cos n\varphi = \frac{\sin \frac{n\varphi}{2} \cos \frac{n+1}{2}\varphi}{\sin \frac{\varphi}{2}}$

\item Доведіть, що якщо $x+iy=(s+it)^{n}$, то $x^{2}+y^{2}=(s^{2}+t^{2})^{n}$

\end{enumerate}
