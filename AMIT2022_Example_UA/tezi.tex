\title{Використання поліномів Бернштейна для моделювання апроксимації функцій.}

\authors{Василь Мельник}

\email{vasyl.melnyk@chnu.edu.ua}

\address{Чернівецький національний університет імені Юрія Федьковича}

В теорії наближення часто використовується поліном Бернштейна для апроксимації функцій, заданих на просторі дійсних чисел. Поліномом Бернштейна $n$~--го порядку для неперервної функції $f(x)\in  СC([0,1])$ називається многочлен вигляду \cite{surname_l_1}:

\begin{equation}
B_{n}(x)=\sum\limits_{k=0}^{n} f(\frac{k}{n}) C_{n}^{k} x^{k} (1-x)^{n-k}. \label{surname_eq_1}
\end{equation}

Можна показати, що $B_{n}(f)$ рівномірно прямує до $f$ на $[0,1]$, тобто, використовуючи їх, можна довести теорему Вейерштрасса (більше про рівномірне наближення в [2]). Цей апроксимаційний метод цікавий тим, що симулює підкидання $n$ разів спеціальної монети, що випадає потрібною нам стороною із ймовірністю $x$.

Цікавим є використання поліномів Бернштейна для узагальнення деяких динамічних рівнянь, зокрема, логістичного відображення:

\begin{equation}
x_{i+1}=rx_{i}(1-x_{i}). \label{surname_eq_2}
\end{equation}

Можна отримати різні варіанти стабілізації (так звані точки стабільності $x_{i}$) для послідовностей $x_{i}$ при різних значеннях $r$. Проте узагальнення вигляду:

\begin{equation}
x_{i+1}(x)=\sum\limits_{k=0}^{n} f(\frac{k}{n}) C_{n}^{k} x_{i}^{k} (1-x_{i})^{n-k}. \label{surname_eq_3}
\end{equation}

дає нам можливість отримати абсолютно іншу картину вигляду точок стабільності, або орбітального паттерну. Зокрема, відображення (3) \label{surname_eq_3} перетворюється в звичайне логістичне відображення (2) \label{surname_eq_2} при $n=2$ та $f(x)=\frac{r}{2}-2r(x-\frac{1}{2})^{2}$. Зокрема, для $n>2$ в деяких випадках зникає еффект періодичного подвоєння орбіт.
