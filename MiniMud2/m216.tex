\section*{Математичний аналіз: границі.}
\end{center}


\textit{\textbf{Теорія.}}  \textit{Послідовність $\{x_{n}\}_{n=1}^{\infty}$ збігається до $a$, якщо для довільного $\varepsilon>0$ існує натуральне число $N$ таке, що для кожного $n>N$ справджується $|x_{n}-a|<\varepsilon$. }

\textit{Позначаємо збіжність через $x_{n}\rightarrow a$ або $\lim\limits_{n\rightarrow \infty}^{}x_{n}=a$.}


\bigskip

Знайдіть наступні границі:

1. $\lim\limits_{x\rightarrow +\infty}^{}\frac{2x^{2}+7x-4}{3-8x+3x^{2}}$

2. $\lim\limits_{x\rightarrow +\infty}^{}\frac{7x^{3}+15x^{2}-7x+9}{5x^{4}-5x^{2}-2x+3}$

3. $\lim\limits_{x\rightarrow +\infty}^{}\frac{2x^{3}+3x-2}{x-1}$

4. $\lim\limits_{x\rightarrow 3}^{}\frac{\sqrt{x+5}-\sqrt{10x-31}}{2x-8}$

5. $\lim\limits_{x\rightarrow 4}^{}\frac{x^{2}-3x-4}{\sqrt{x+5}-3}$

\bigskip

Вавилонський метод обчислення $\sqrt{2}$.

6. Нехай $x_{1}=1$, $x_{n+1}=\frac{1}{2}(x_{n}+\frac{2}{x_{n}})$ при $n\geq 1$. Доведіть, що $\lim\limits_{n\rightarrow +\infty}^{}x_{n}=\sqrt{2}$.

\textit{\textbf{Означення.}} \textit{Числом $e$ називається границя послідовності $x_{n}=(1+\frac{1}{n})^{n}$.}

\bigskip

Знайдіть $\lim\limits_{n\rightarrow +\infty}^{}x_{n}$:


7. $x_{n}=(1-\frac{1}{n})^{n}$

8. $x_{n}=\frac{2n^{2}-5n+3}{1+2n+3n^{2}}$

9. $x_{n}=\frac{7n^{3}+10n^{2}+n+1}{10n^{4}-3n^{2}+3n-3}$

10. $x_{n}=\sqrt{n+1}-\sqrt{n}$
