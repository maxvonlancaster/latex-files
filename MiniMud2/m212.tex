\section*{Функції однієї змінної. Функціональні рівняння.}
\end{center}

1. Знайдіть всі функції $f(x)$ визначені для всіх дійсних $x$, такі, що виконується рівність $2f(x)+f(1-x)=x^{2}.$

2. Знайдіть всі такі функції $f(x)$, що для довільного $x$ виконується: $f(2x+1)=4x^{2}+14x+7.$

3. Нехай для функції $f$ та довільних чисел $x$ та $y$ виконується $f(xy)=f(x)+f(y).$ Знайти $f(2007)$, якщо $f(\frac{1}{2007})=1$.

4. Нехай для функції $f$ та довільного $x$ виконується $f(x)+(x+\frac{1}{2})f(1-x)=1.$

а) Знайти $f(0)$ і $f(1)$.

б) Знайти $f(x)$.

5. Нехай для функції $F$ та довільного $x$ виконується $F(x+1)F(x)+F(x+1)+1=0.$ Довести, що $F$ не може бути неперервною.

6. Знайти всі функції $f$, такі, що для довільних $x,y,z$ виконується нерівність $f(x+y)+f(y+z)+f(z+y)\geq 3f(x+2y+3z).$

7. Для яких дійсних $\alpha$ існує функція $f$ не рівна константі, така, що для довільних $x,y$ виконується $f(\alpha(x+y))=f(x)+f(y).$

8. Нехай функція $f(x)$ визначена для довільного $x$ та при $x\neq 1$ виконується $$(x-1)f(\frac{x+1}{x-1})-f(x)=x.$$ Знайдіть а) $f(-1)$; б) $f(x)$.

9. Нехай функція $f(x)$ задана для всіх дійсних чисел, та для довільних $x,y$ при $x>y$ виконується $(f(x))^{2}\leq f(y).$ Доведіть,  що множина всіх значень функції $f$ належить відрізку $[0,1]$
