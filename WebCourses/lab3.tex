\section*{Лабораторна 3.}

\begin{enumerate}
\item Додайте сторінки реєстрації та логіну до вашої фронтенд аплікації.

\item Додайте ендпоінт для реєстрації нового користувача, та ендпоінт для аутентифікації користувача, що вертатиме JWT беарер токен.
    
\item Додайте перевірку на наявність беарер токена в запиті до хоча б одного із старих ендпоінтів. Передбачте повідомлення помилки 403 для користувача.
    
\item* \textit{(для бажаючих)} Додайте OAuth2 аутентифікацію до вашої аплікації.

\item Додайте вебсокет до вашої аплікації. Достатньо простого повідомлення з певним інтервалом. При бажанні, можете додати нотифікацію для користувача при певній події (наприклад, інший користувач редагує ресурс, з яким працює даний користувач).

\item* \textit{(для бажаючих)} Додайте graphql квері ендпоінт до бекенд частини аплікації.

\item Оформіть завдання данної лабораторної як PR, та надайте посилання на нього в moodle.

\end{enumerate}
