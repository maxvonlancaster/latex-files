\section*{Математична індукція.}
\end{center}

\medskip

\hrule height 1pt
\vskip 3pt \hrule

\medskip

\textit{\textbf{Математична індукція} ~--- Твердження $A(n)$ , залежне від натурального параметра $n$ , вважається доведеним для всіх натуральних $n$ , якщо воно доведене для $n =1$ і з припущення про виконання твердження $A(k)$ для якого-небудь натурального $k$ виведено, що справджується твердження $A(k+1)$ для натурального числа $k+1$.}

\textit{Яскравою ілюстрацією принципу математичної індукції є картина падіння ряду доміно. Достатньо падіння першого доміно, та того, щоб кожне доміно своїм падінням перевернуло наступне, щоб весь ряд впав.}

\medskip

\end{center}

\medskip



\begin{enumerate}
\item Доведіть, що для довільного натурального $n$ число $2^{2n}-1$ ділиться на 3.

\item Числа $N_{1}$, $N_{2}$, … $N_{n}$ є сумами квадратів двох цілих чисел. Доведіть, що і їх добуток є сумою квадратів двох цілих чисел.

\item Доведіть, що для довільного натурального $n$ число $5\cdot 2^{3n-2}+3^{3n-1}$ ділиться на 19.

\item Декілька прямих розбивають площину на частини. Доведіть, що ці частини можна розфарбувати в два кольори так, що ті частини, що межують, матимуть різний колір.

\item Вершини опуклого багатокутника зафарбовані в три кольори так, що кожен колір є присутнім і жодні дві сусідні вершини не зафарбовані в один колір. Доведіть, що багатокутник можна розбити діагоналями на трикутники так, щоб у кожного трикутника вершини були трьох різних кольорів.

\item Нехай в шахматній дошці $2^{n}\times 2^{n}$ видалено кутову клітинку. Доведіть, що цю фігуру можна повністю покрити, без перекриття, фігурками вигляду:

\includegraphics[scale=0.3]{triomino.png}

\item Нехай $n\in \mathbb{N}$ хорд проведено всередині кола так, що кожні дві хорди перетинаються, але жодні три не перетинаються в одній точці. Доведіть, що коло розбите хордами на $\frac{n^{2}+n+2}{2}$ регіони.


\item Правильний трикутник зі стороною $2^{n}$ розділено на правильні трикутники зі стороною $1$, та видалено верхній трикутник.

    \includegraphics[scale=0.8]{tr1.png}

    Доведіть, що отриману фігуру можна повністю покрити фігурами (без перекриття), що складаються кожна із $3$-х трикутників вигляду:

    \includegraphics[scale=0.8]{tr2.png}

\item Доведіть, що квадрат можна розбити на $n$ квадратів, для довільного $n\geq 6$.

\item Нехай на колі вибрано $n+2$ точок, і вони послідовно з'єднані відрізками, кожна точка із двома сусідніми. Доведіть, що сума кутів утвореного $n+2$-кутника буде рівною $180n^{\circ}$.

\item Доведіть, що $$\frac{1}{1\cdot 2\cdot3} + \frac{1}{2\cdot 3\cdot4}+...+\frac{1}{n(n+1)(n+2)} = \frac{n(n+3)}{4(n+1)(n+2)}.$$

\item Послідовність $\{a_{n}\}$ така, що $a_{1} = 3$, $a_{2} = 5$, $a_{n+2} = 3a_{n+1}- 2a_{n}$. Доведіть, що $a_{n} = 2^{n}+ 1$ для всіх натуральних $n$.

\item Число $x+\frac{1}{x}$ є цілим. Доведіть, що при кожному натуральному $n$ число $x^{n}+\frac{1}{x^{n}}$~--- також ціле.

\item Доведіть, що при всіх натуральних $n \geq 2$ виконується нерівність $2^{n}>1+n\sqrt{2^{n-1}}$.

\item Довести рівність для всіх натуральних $n$: $$\cos \alpha \cdot \cos 2\alpha\cdot \cos 4\alpha\cdot \ldots \cdot \cos 2^{n}\alpha = \frac{\sin 2^{n+1}\alpha}{2^{n+1}\sin \alpha}$$

\item Доведіть рівність $$\underbrace{\sqrt{2+\sqrt{2+...+\sqrt{2+\sqrt{2}}}}}_{n}=2\cos\frac{\pi}{2^{n+1}}$$

\end{enumerate}
