\section*{Деякі олімпіадні задачі.}
\end{center}

\bigskip


\hrule height 1pt
\vskip 3pt \hrule
\medskip
\medskip

\medskip


\begin{enumerate}
\item Доведіть, що якщо $a=b-1$, то $$(a+b)(a^{2}+b^{2})(a^{4}+b^{4})\ldots (a^{32}+b^{32})=a^{64}-b^{64}$$

\item Знайдіть всі пари дійних $(x,y)$, що задовільняють $$\{x\}+\{y\}=[x+y].$$

\item Знайдіть усі дійсні числа $x$, для яких виконується рівність $$\{3 \{3 \{3 \{3 \{3x\} \} \} \} \} = x .$$

\item Натуральне число $k >1$. Знайдіть усі пари цілих чисел $x, y$, які задовольняють рівність $y^{k}=x^{2}+x.$

\item Сума кількох послідовних натуральних чисел (більше одного) дорівнює 2017. Знайдіть усі набори таких чисел.

\item Для додатних чисел $a$, $b$, $c$ доведіть нерівність: $$(a+1)^{2}+(b+1)^{2}+(c+1)^{2}\geq 3(1+a\sqrt[3]{b} + b\sqrt[3]{c} + c\sqrt[3]{a}).$$

\item Знайдіть усі трицифрові натуральні числа $N$, які дорівнюють сумі цифр числа $N$, до якої додається куб суми цифр числа $N$.

\item Знайдіть усі прості числа $p$ такі, що $p^{5}-p = 2n!.$ 

\item Нехай $a,b,c\geq0$. Доведіть, що $ab+bc+ca\geq a\sqrt{bc}+b\sqrt{ac}+c\sqrt{ab}$.

\end{enumerate}
